# BrightBook Library Platform

BrightPath Library Platform is a lightweight digital library system designed for schools and community centers. It provides essential tools for users to access library services through a clean and simple interface.

## Features
- User login and signup  
- Catalog search (database-backed)  
- Digital library card with QR code  
- Local event listings  
- Basic room reservations  
- Friendly layout  

## Tech Stack
- Frontend: React 
- Backend: Spring Boot
- Database: MySQL
- Deployment: AWS
