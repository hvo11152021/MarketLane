## Seller User Stories

### US-S1: Add Products
**As a** Seller  
**I want to** add new products to the store  
**So that** buyers can purchase them.

### US-S2: Edit Products
**As a** Seller  
**I want to** update product information  
**So that** the product listings stay accurate.

### US-S3: Remove Products
**As a** Seller  
**I want to** remove products  
**So that** unavailable or outdated items are not shown to buyers.

### US-S4: Manage Inventory
**As a** Seller  
**I want to** adjust product stock levels  
**So that** buyers see correct availability.

### US-S5: View Orders
**As a** Seller  
**I want to** view all incoming orders  
**So that** I can fulfill them.

### US-S6: Update Order Status
**As a** Seller  
**I want to** update the order status (e.g., processing, shipped)  
**So that** buyers can track their purchases.

### US-S7: View Sales Reports
**As a** Seller  
**I want to** see sales summaries  
**So that** I can understand business performance.

## Buyer User Stories

### US-B1: Browse Products
**As a** Buyer  
**I want to** browse available products  
**So that** I can discover items to purchase.

### US-B2: Search Products
**As a** Buyer  
**I want to** search for products  
**So that** I can quickly find what I need.

### US-B3: View Product Details
**As a** Buyer  
**I want to** see detailed information about a product  
**So that** I can make an informed decision.

### US-B4: Add to Cart
**As a** Buyer  
**I want to** add products to my cart  
**So that** I can buy multiple items at once.

### US-B5: Checkout and Pay
**As a** Buyer  
**I want to** checkout and make a payment  
**So that** I can complete my purchase.

### US-B6: View Order Status
**As a** Buyer  
**I want to** track my order  
**So that** I know when it will arrive.

### US-B7: Review Products
**As a** Buyer  
**I want to** leave product reviews  
**So that** other buyers can benefit from my experience.

## Admin User Stories

### US-A1: Manage Users
**As an** Admin  
**I want to** manage user accounts (activate/deactivate)  
**So that** the system remains secure and orderly.

### US-A2: Manage Roles & Permissions
**As an** Admin  
**I want to** assign roles (buyer, seller, admin)  
**So that** each user has the correct access.

### US-A3: Monitor System Activity
**As an** Admin  
**I want to** view logs and system activity  
**So that** I can ensure stability and detect issues.
